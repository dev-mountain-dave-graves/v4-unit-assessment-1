HTML5 is a mark up language used for structuring and presenting content on the World Wide Web, It is the fifth and latest major version of HTML that is a World Wide Web Consortium (WC3) recommendation. The current specification is known as the HTML Living Standard and is maintained by a consortium of the major browser vendors (Apple, Google, Mozilla, and Microsoft), the Web HypertextApplication TechnologyWorking Group (WHATWG).
